# Webpack + Express out-of-box boilerplate

## Developer Benefits
* React.js Compatible
* Old fashion page Based JS/CSS Compatible
* SASS Hot Reload
* Production/Development Settings in one file
* Bootsrap, Fontawesome, jQuery enabled

### TBA

## Getting Started
### Development
```
npm nn


npm start
```

### Production

```
npm run build
NODE_ENV=production npm start
```
